# 書籍マークダウン要約スキル

このスキルは、書籍の一覧（行ごと、CSV、表形式など）を受け取り、各書籍について以下のフィールドを含むマークダウン形式の要約を生成します。

出力フィールド（順序）:
- タイトル
- 著者
- あらすじ（短め: 1～3文）
- 発売年

入力の受け取り方（柔軟）:
- 各行が「タイトル, 著者, 発売年, あらすじ」のCSV
- タブ区切りやスペースで並んだ列
- 単純な箇条書き（解析して推測）

振る舞い:
- 不完全なデータがある場合、欠損フィールドは「不明」や空欄ではなく簡潔に推測または「不明」と明示する。
- あらすじは要点を押さえ短くまとめる（1～3文）。
- 出力はUTF-8のマークダウン。各書籍を見出し（###）で区切る。

出力例:

### 『サンプルタイトル』
- 著者: 山田 太郎
- 発売年: 2020
- あらすじ: 近未来を舞台にした冒険物語。主人公が困難を乗り越えて成長する過程を描く。

## 使用例

入力 (CSV形式):
```
サンプルタイトル,山田 太郎,2020,近未来を舞台にした冒険物語。主人公が困難を乗り越えて成長する過程を描く。
別の本,鈴木 花子,2018,経済学の入門書。分かりやすい例で概念を説明する。
```

上記を投げると、各書籍ごとに上のようなマークダウンを生成します。

注意事項:
- 機密情報や著作権で保護された全文は要約対象にしないでください。短いメタデータや要約は問題ありません。
- 生成スタイルは一貫させること。必要ならユーザーに要約の長さ（短め/中程度/長め）を尋ねるプロンプトを促す。

# テスト用プロンプト例
- 私の書籍リストをマークダウンで要約して
- 書籍一覧を要約して
